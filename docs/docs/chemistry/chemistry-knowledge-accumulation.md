# 化学积累

1. 石墨有吸附功能, 石墨电极可以吸附生成的气体.
2. $Co^{3+}$ 的氧化性极强 (大于 $Cl_2/ClO_3^-$). 
3. 见到 $Fe^{3+}/Al^{3+}$ 时要想强酸性环境以及相互促进水解. 
4. 见到浓硫酸想难电离与遇水放热.
5. 除 $Ca^{2+}/Mg^{2+}$ : 转化为 $CaF_2/MgF_2$.
6. 还原性顺序: $S^{2-} > I^- > Fe^{2+} > Br^- > Cl^- > Mn^{2+}$ , 留点铁修路吗.
7. 铁氰化钾滴入有铁单质的溶液中(如原电池铁电极), 铁氰化钾中的三价铁会和铁单质反应生成二价铁从而出现蓝色沉淀.

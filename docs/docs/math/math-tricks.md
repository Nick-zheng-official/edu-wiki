## 三角变换

1. 三角函数 相加/相乘 想和差化积/积化和差

## 不等式

1. $x_1x_2 < x_1 + x_2$, 看见和与积可能是
   - 完全平方式/基本不等式
   - __因式分解__
2. 想要加法变乘法(如$f(x) = e^x + e^{-x} + \dots \ge 2\sqrt{e^x \cdot e^{-x}} + \dots = 2 + \dots $), 在指数特别复杂且发现倒数不需要精确值(如证明不等式或导函数正负判断)的时候可以使用(一般题目会给使用处的条件引导)

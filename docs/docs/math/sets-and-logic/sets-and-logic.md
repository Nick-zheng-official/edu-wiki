# 集合与逻辑用语

## 集合

集合是研究对象所构成的集体, 一般用大写字母表示, 而元素是构成集合的研究对象, 一般用小写字母表示. 

集合有三个特性:
1. 确定性: 集合的元素必须是确定的
2. 无序性: 集合中的元素可以任意排列
3. 互异性: 集合中的元素必须是不同的

数集:
- $\mathbb{N}$: 自然数集
- $\mathbb{N_+/N^*}$: 正整数集
- $\mathbb{Z}$: 整数集
- $\mathbb{Q}$: 有理数集
- $\mathbb{R}$: 实数集
- $\mathbb{C}$: 复数集

$\in$ 为属于符号. 

可以用列举法和描述法表示集合. 列举法即列出所有元素, 如 $\{1, 4, 5\}$ , 描述法形如 $\{研究对象|特征\}$ , 如 $\{x\in\mathbb{Z} | 1 < x < 5\}$ . 若未知数 $x \in \mathbb{R}$ , 则一般可以省略. 

$\varnothing$ 为空集, 此符号即代表一个集合, 此集合内没有任何元素. 注意 $\{\varnothing\}$ 并非空集, 此集合中包含空集. 当然, 集合的嵌套是允许的. 

取值范围需要写成集合或区间的形式. 

子集: 若集合 $A$ 中任一元素均属于 $B$ , 那么 $A$ 为 $B$ 的子集, 即 $A \subseteq B$ . 

使用 $Venn$ 图或数轴表示集合关系十分实用.  

集合 $A = B$ 等价于 $A \subseteq B, B \subseteq A$ . 特殊地, 空集是任一集合的子集. 真子集即当 $A \subseteq B, A \ne B$ 时 $A$ 为 $B$ 的真子集, 表示为 $A \subsetneqq B$ . 集合问题考虑空集. 列不等式先考虑是否取等. 

子集个数问题首先确定集合的元素个数 $n$ , 子集个数即为 $2^n$ 个(每个元素有存在/不存在两种状态), 非空子集, 真子集, 非空真子集个数易得. 

求 $A$ 与 $B$ 的交集 $C$ (且), 表示为 $C = A \cap B$ ; 求 $A$ 与 $B$ 的并集 $C$ (或), 表示为 $C = A \cup B$ ; 
求 $A$ 在全集 $U$ 中的补集 $B$ (非), 表示为 $B = \complement_U A$ .

## 逻辑用语

$条件 \xrightleftharpoons[必要性]{充分性} 结论$ ,  由此可以引出既不充分也不必要条件, 充分不必要条件, 必要不充分条件, 充分必要条件(充要条件). 

注意题目的问法, 问 $P$ 是 $Q$ 的什么条件, 与问 $Q$ 的什么条件是 $Q$ , 有时可能看反, 只需要先找 "条件是谁" 或 "谁是条件" 即可理解, 条件放左边, 结论放右边. 

小范围可以推大范围, 大范围不能推小范围. 通过 $Venn$ 图和数轴可以形象理解. 

$\exists$ 存在量词表示存在; $\forall$ 全称量词表示所有. 命题否定形式 $\lnot p$ 对于 $p$ 需要更改量词, 并将结论取反, 条件不作修改. $p$ 与 $\lnot p$ 一真一假, 故出现假命题考虑用否命题转化为真命题.

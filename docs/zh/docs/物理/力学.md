# 力学

力学是研究物体机械运动规律的科学，是物理学的基础分支。

## 质点运动学

### 基本概念

- **质点**：忽略物体大小和形状，只考虑其质量的理想化模型
- **参考系**：描述物体运动时所选的参照物
- **坐标系**：定量描述物体位置的数学工具

### 运动的描述

- **位置矢量**：从原点指向质点位置的矢量，用 \( \vec{r} \) 表示
- **位移**：质点位置的变化，\( \Delta \vec{r} = \vec{r}_2 - \vec{r}_1 \)
- **速度**：描述物体运动快慢和方向的物理量
  - 平均速度：\( ar{v} = \frac{\Delta \vec{r}}{\Delta t} \)
  - 瞬时速度：\( \vec{v} = \lim_{\Delta t 	o 0} \frac{\Delta \vec{r}}{\Delta t} = \frac{d\vec{r}}{dt} \)
- **加速度**：描述速度变化快慢的物理量
  - 平均加速度：\( ar{a} = \frac{\Delta \vec{v}}{\Delta t} \)
  - 瞬时加速度：\( \vec{a} = \lim_{\Delta t 	o 0} \frac{\Delta \vec{v}}{\Delta t} = \frac{d\vec{v}}{dt} \)

## 牛顿运动定律

### 牛顿第一定律（惯性定律）

任何物体都要保持匀速直线运动或静止状态，直到外力迫使它改变运动状态为止。

### 牛顿第二定律（加速度定律）

物体的加速度与所受合外力成正比，与物体的质量成反比，加速度的方向与合外力的方向相同。

数学表达式：\( \vec{F} = m\vec{a} \)

### 牛顿第三定律（作用力与反作用力定律）

两个物体之间的作用力和反作用力总是大小相等，方向相反，作用在同一条直线上。

数学表达式：\( \vec{F}_{12} = -\vec{F}_{21} \)

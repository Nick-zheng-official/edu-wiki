# Edu-Wiki

欢迎来到Edu-Wiki，一个专注于高中知识的维基百科。

## 关于我们

Edu-Wiki是一个开源的高中知识平台，致力于为高中生提供高质量、结构化的学习资源。

## 主要内容

- **数学**：代数、几何、三角函数、微积分等
- **物理**：力学、电磁学、热学、光学等
- **化学**：无机化学、有机化学、化学实验等
- **生物**：细胞生物学、遗传学、生态学等
- **语文**：文言文、现代文、作文写作等
- **英语**：语法、词汇、阅读理解等

## 如何使用

1. 使用顶部导航栏选择学科
2. 在左侧边栏浏览具体知识点
3. 使用搜索功能查找特定内容
4. 点击页面右上角的GitHub图标参与贡献

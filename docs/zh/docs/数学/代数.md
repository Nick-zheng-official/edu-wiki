# 代数

代数是数学的一个分支，研究数字、符号和它们之间的关系。

## 基本概念

### 变量与常量

- **变量**：可以取不同值的符号，通常用字母表示，如 \( x, y, z \)
- **常量**：固定不变的值，如 \( 1, 2, 3 \) 或 \( \pi \)

### 代数式

代数式是由变量、常量和运算符号组成的表达式，如：
- \( 2x + 3 \)
- \( x^2 - 5y + 1 \)
- \( \frac{a + b}{c} \)

## 方程与不等式

### 一元一次方程

形式：\( ax + b = 0 \)（其中 \( a 
eq 0 \)）

解法：\( x = -\frac{b}{a} \)

示例：
\( 2x + 5 = 0 \)
解得：\( x = -\frac{5}{2} \)

### 一元二次方程

形式：\( ax^2 + bx + c = 0 \)（其中 \( a 
eq 0 \)）

解法：
- 配方法
- 公式法：\( x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \)
- 因式分解法

## 函数

### 一次函数

形式：\( y = kx + b \)（其中 \( k, b \) 为常数，\( k 
eq 0 \)）

图像：一条直线，斜率为 \( k \)，截距为 \( b \)

### 二次函数

形式：\( y = ax^2 + bx + c \)（其中 \( a, b, c \) 为常数，\( a 
eq 0 \)）

图像：抛物线，开口方向由 \( a \) 的正负决定

## 数列

### 等差数列

定义：从第二项起，每一项与前一项的差为常数的数列

通项公式：\( a_n = a_1 + (n - 1)d \)

前 \( n \) 项和公式：\( S_n = \frac{n(a_1 + a_n)}{2} = na_1 + \frac{n(n - 1)}{2}d \)

### 等比数列

定义：从第二项起，每一项与前一项的比为常数的数列

通项公式：\( a_n = a_1 q^{n - 1} \)

前 \( n \) 项和公式：\( S_n = \frac{a_1(1 - q^n)}{1 - q} \)（其中 \( q 
eq 1 \)）

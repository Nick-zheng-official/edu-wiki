## 介绍

制作背景, 目的

随着越来越多的老师在网络上分享自己的基础教育资源, 我们开始思考是否有一种方式, 能够将这些资源汇总起来, 形成一个统一的平台, 以便于学生学习, 同时解决纸质资源不方便携带的问题, 并方便学生在任何地方任何时间进行复习。

因此，**EDU Wiki** 被创建了。

发展历程

内容与待完善部分

发展方向

此外，鉴于以下两点

- 基础教育资源应该以开放的方式分享
- 解题技巧在快速迭代更新，在面对新的题目时，旧的技巧也随时可能失效

因此，**EDU Wiki** 永远不会出版书籍。

最后，**EDU Wiki** 源于社区，作为**独立的组织**，提倡**知识自由**，在未来也绝不会商业化, 亦不会出版任何书籍，将始终保持**独立自由**的性质。

## 部署

本项目目前采用 [mkdocs](https://github.com/mkdocs/mkdocs) 部署在 [https://edu-wiki.pages.dev/](https://edu-wiki.pages.dev/). 

 本项目可以直接部署在本地, 具体方式如下:
 
 ```bash
# 克隆仓库
git clone https://github.com/Nick-zheng-official/edu-wiki.git
cd edu-wiki

# 安装依赖
pip install -r requirements.txt

# 启动本地预览
mkdocs serve
```

启动后访问 <http://127.0.0.1:8000/> 即可查看网站。mkdocs 本地部署的网站是动态更新的，即当你修改并保存 md 文件后，刷新页面就能随之动态更新。
